# 高岛易断 - 卦象呈现系统

基于高岛吞象流的易经占卜辅助工具，支持手动输入和自动起卦两种模式。

## 功能特性

### 手动输入模式
直接输入三个数字完成起卦：
- **上卦数** (1-8)：先天八卦数
- **下卦数** (1-8)：先天八卦数
- **动爻** (1-6)：变爻位置

### 自动起卦模式（数字分蓍法）
模拟传统49签分蓍起卦仪式，通过三次"长按-松开"交互完成起卦。

#### 起卦流程
1. **选择性别** - 决定取签规则（男左女右）
2. **第一签·求上卦** - 男取左手，女取右手，除8取余
3. **第二签·求下卦** - 男取右手，女取左手（相反），除8取余
4. **第三签·求动爻** - 男取左手，女取右手，除6取余

#### 交互设计
- **长按按钮**：模拟屏住呼吸、凝神静气
- **松开按钮**：模拟手中签一分为二的瞬间
- **最低按压500ms**：确保足够的凝神时间

## 高岛流断卦权重

- **本卦**：30% - 事情的本质状态
- **动爻**：51% - 关键变化因素（最重要）
- **变卦**：19% - 最终结果走向

## 技术实现

- React 18 + Tailwind CSS
- 单文件 HTML，可直接在浏览器中运行
- 响应式设计，支持移动端和桌面端

### UI/UX 特性

- **深色/浅色模式**：自动检测系统偏好，支持手动切换，记忆用户选择
- **无障碍设计**：WCAG 4.5:1 对比度标准，ARIA 标签，键盘导航支持
- **动画优化**：支持 `prefers-reduced-motion` 减少动画
- **中文优化**：针对中文字符的高对比度设计
- **SVG 图标**：使用 Heroicons 而非 emoji，确保视觉一致性

## 传统分蓍法原理

本工具的自动起卦功能模拟了传统的蓍草（竹签）分卦方法：

1. 50签去1，留49签
2. 凝神静气，默念所问之事
3. 深呼吸，气息调匀
4. 在屏息与吐气的瞬间，将签一分为二
5. 根据性别取左手或右手的签数
6. 计算余数得到卦数或爻位

此方法结合了梅花易数与蓍草法的特点，强调"心念控制呼吸，呼吸控制双手，双手控制分蓍"的心法。
